# Session Overview

## Goal

Create a canonical journey-traceability standard for Phenotype repos so docs
carry keyframes and recordings, not just prose.

## Success Criteria

- A governance doc exists for the journey/keyframe/recording contract.
- The standard points to the hwLedger pattern as the reference shape.
- The existing governance docs link to the new standard.
- The session documents record the rationale and rollout steps.

## Scope

- Centralize the journey artifact contract in `phenotype-infra`.
- Keep the standard repo-agnostic so it can be reused across all repos.
- Add the standard to the local governance surface.
