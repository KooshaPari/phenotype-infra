# Architecture Decision Records

This directory contains the Architecture Decision Records (ADRs) for this project, following the [MADR](https://adr.github.io/madr/) template.

## Index

- [0001 — Record architecture decisions](0001-record-architecture-decisions.md)

## How to add a new ADR

1. Copy `0001-record-architecture-decisions.md` to a new file with the next sequential number (e.g., `0002-choose-database.md`).
2. Update the title and content.
3. Update this index with a link to the new ADR.
4. Open a PR.
