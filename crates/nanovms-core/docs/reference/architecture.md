## Module Architecture

```
nanovms/
├── cmd/                    # CLI entry points
│   ├── nanovms/          # Main CLI
│   └── nanovmsd/          # Daemon/server
├── internal/
│   ├── api/              # gRPC and REST API
│   │   ├── v1/          # API version 1
│   │   └── proto/        # Protocol buffers
│   ├── adapters/         # Platform implementations
│   │   ├── vm/           # VM adapters (Firecracker, QEMU, etc.)
│   │   ├── sandbox/      # Sandbox adapters (gVisor, bwrap, etc.)
│   │   └── wasm/         # WASM adapters (Wasmtime, WAMR)
│   ├── core/             # Core business logic
│   │   ├── scheduler/    # VM/sandbox scheduling
│   │   ├── lifecycle/    # Lifecycle management
│   │   ├── resource/     # Resource allocation
│   │   └── quota/         # Quota management
│   ├── domain/           # Domain models
│   │   ├── vm/          # VM entities
│   │   ├── sandbox/      # Sandbox entities
│   │   └── wasm/         # WASM entities
│   └── ports/            # Interface definitions
├── pkg/                   # Shared packages
│   ├── config/           # Configuration management
│   ├── crypto/           # Cryptography utilities
│   ├── fs/               # Filesystem utilities
│   ├── net/              # Networking utilities
│   └── vm/               # VM utilities
├── sdk/                   # Language SDKs
│   └── rust/              # Rust SDK (nvms-sdk)
│       ├── src/           # Library source
│       └── Cargo.toml     # Manifest
├── integrations/          # External integrations
│   └── pheno-compose/     # PhenoCompose driver
└── tools/                 # Development tools
    ├── benchmark/         # Benchmark suite
    └── fuzz/              # Fuzzing tests
```
